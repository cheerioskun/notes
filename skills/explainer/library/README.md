# Component library

Reusable building blocks for explainer posts. See `../SKILL.md` → *Augmenting this skill* for the rules.

One line per component.

| Component | Purpose | First used |
|---|---|---|
| [theme-toggle](theme-toggle.html) | Explicit light/dark switch via `html[data-theme]` + `localStorage`. | GIL Part 1 |
| [reading-progress](reading-progress.html) | Fixed 2px scroll-position bar. Use only on posts >10 min. | GIL Part 1 |
| [toc](toc.html) | Numbered (decimal-leading-zero) two-column TOC. Use when post has 7+ sections. | GIL Part 1 |
